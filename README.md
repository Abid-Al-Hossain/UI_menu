# UI Foundry Menu Studio

Standalone, self-contained Menu component generator built for marketplace sale and future UI Foundry SaaS merge-readiness.

## Features

- Navigation model: items, current state, disabled state, separators, groups, nested levels
- Keyboard behavior: roving focus, typeahead, arrow keys where APG/Radix patterns apply
- Responsive behavior: collapsed/mobile modes, overflow and alignment
- Accessibility: nav labels, aria-current, menu roles, step state labels
- React-only export: preview, visible code, copy, and download are sourced from one payload.
- Local shared controls: no runtime imports from other component projects.

## Commands

```bash
npm install
npm run typecheck
npm run lint -- --quiet
npm run build
npm run dev
```
